## What is PassGym       

- 프로젝트 소개

  PassGym 프로젝트는 사용자와 피트니스, 헬스장을 연결해주는 플랫폼 프로젝트 입니다.
  사용자는 자신의 위치와 가까운 헬스장, 본인이 찜한 헬스장 등을 한눈에 파악할 수 있습니다. 
  
  
- 프로젝트에 사용한 기술
   - JAVA
   - Oracle Data Base
   - servlet
   - JSP
   - JavaScript
   - Jquery
   - HTML/CSS
   - githup(프로젝트 버전 관리)

- 프로젝트에 사용한 디자인 패턴   

   MVC 패턴 사용
   
- 프로젝트에 사용된 Open API
  - 카카오맵 API
